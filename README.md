#Anime-Style Interactive Visual Novel Website 

一个偏视觉小说风格的 AI 角色对话网页项目。

## 功能

- 角色人设编辑
- AI 对话与流式输出
- 对话记录与日志面板
- 角色卡导入 / 导出
- 背景图、立绘、BGM 管理
- 语音播放与音量控制
- Mock / Live API 切换

## 技术栈

- HTML
- CSS
- JavaScript

## 目录结构

```text
index.html
styles.css
app.js
README.md
```

## 使用方法

1. 直接用浏览器打开 `index.html`
2. 在设置面板中填写角色信息和 API 配置
3. 选择 `Mock` 或 `Live API` 模式
4. 输入内容并发送对话

## 说明

- `app.js` 负责核心交互逻辑
- 页面状态会保存在浏览器本地
- API Key 仅保存在本地浏览器，不会自动上传

## 备注

本项目开发过程中使用了 AI 辅助完成部分功能与优化。
